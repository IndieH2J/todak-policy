# 토닥(Todak) 계정 및 데이터 삭제 요청 안내

**시행일**: 2026년 5월 21일

토닥은 이용자가 본인의 모든 데이터를 언제든지 삭제할 수 있도록 두 가지 경로를 제공합니다.

---

## 1. 앱 내에서 즉시 삭제 (권장 · 가장 빠름)

토닥 앱이 설치되어 있다면 앱 내 기능으로 **모든 데이터를 즉시 셀프 삭제**할 수 있습니다.

1. 토닥 앱 실행
2. 하단 탭의 **설정** 진입
3. **기타 > 데이터 초기화** 선택
4. 확인 다이얼로그 → 삭제 진행

위 절차는 단말의 로컬 데이터와 Firebase 의 익명 UID 에 묶인 서버 데이터(감정 기록 · 일기 · 사용자 정의 행동 · FCM 토큰)를 **모두 즉시 삭제**합니다. 추가 처리 시간이 없습니다.

---

## 2. 앱을 사용할 수 없는 경우 — 이메일 요청

앱을 삭제했거나 단말이 분실되어 앱 내 기능을 사용할 수 없는 경우, 이메일로 삭제를 요청해 주세요.

**연락처**: <indieh2j@gmail.com>
**제목 예시**: `[토닥] 데이터 삭제 요청`

이메일 본문에 다음 정보를 포함해 주세요:

- 토닥 익명 사용자 UID (앱 내 설정 화면에서 확인 가능했던 경우)
- UID 를 모를 경우: 토닥을 마지막으로 사용한 대략적인 날짜·기록 내용 일부 (운영자가 식별을 도와 처리할 수 있도록)

운영자는 본인 확인 후 **요청 접수일로부터 10일 이내** 에 삭제를 완료하고 결과를 회신합니다.

---

## 삭제되는 데이터

| 항목 | 저장 위치 |
|---|---|
| 익명 사용자 UID | Firebase Authentication |
| 감정 기록 (감정 카테고리 · 기록 시각 · 원인 텍스트) | Firestore `emotion_logs` |
| 행동(해결책) 선택 기록 | Firestore `log_actions` |
| 회고 일기 | Firestore `daily_diaries` |
| 사용자 정의 해결책 | Firestore `users/{uid}/custom_actions` |
| FCM 디바이스 토큰 | Firestore `users/{uid}/fcm_tokens` |
| 단말 로컬 캐시 · PIN · 암호화 키 | 디바이스 Secure Storage |

## 보관 예외

- **없음.** 토닥은 법령에 의한 보관 의무가 발생하는 데이터(거래 내역 · 결제 정보 등)를 수집하지 않으므로, 삭제 요청 시 *예외 없이 전부 삭제* 합니다.

---

## 자동 삭제 (참고)

이용자의 명시적 요청 없이도 다음 경우 자동 삭제가 발생합니다.

- 앱 내 PIN 인증 **15회 연속 실패** 시 단말의 모든 로컬 데이터 자동 삭제 (보안 정책)
- 단말에서 앱을 제거하면 단말의 로컬 데이터(PIN · 암호화 키 등)는 즉시 제거 (단, Firebase 의 서버 데이터는 그대로 유지 — 별도 요청 필요)
